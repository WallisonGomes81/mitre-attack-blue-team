# T1047 - Windows Management Instrumentation (WMI)

## Objetivo
Detectar uso indevido do WMI para movimentação lateral ou execução remota.

## Descrição da Técnica
A técnica T1047 permite execução remota de comandos via WMI, frequentemente usada para movimentação lateral.

## Fonte de Log
- Windows Event ID 4688
- WMI-Activity Operational Log (Event ID 5857, 5858)
- Sysmon Event ID 1

## Indicadores de Comprometimento
- Execução de wmiprvse.exe iniciando processos suspeitos
- Conexões remotas via WMI
- Criação de processos remotos sem interação do usuário

## Severidade
Média a Alta

## Ação Recomendada
- Monitorar conexões remotas
- Validar legitimidade da administração remota
- Aplicar segmentação de rede
